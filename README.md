# Jasmine + Karma + Typescript + Webpack

This build typescript by webpack and setup testing with Jasmine and Karma via typescript. This rep is example how build this tools to one system.

  - TypeScript
  - Webpack
  - Jasmine
  - Karma
  - Babel
  - async/await
  - sourcemap
  - Multi-browsers
